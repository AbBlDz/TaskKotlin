------------------
Тестовые сценарии
------------------

Тест 1. Передача данных программе с запуском.
1) Запустить командную строку (cmd)
2) Перейти в директорию с программой (>cd ..\TestTask)
3) Запустить скрипт сборки программы (BUILD.bat)
4) Подождать пока скрипт скомпилирует и соберет файл приложения.
5) Передать текст программе как переменную (java -jar TaskKotlin.jar [текст])
6) Запустить программу
7) Получить результат в командной строке

Тест 2. Передача данных через стандартный поток ввода.

2) Перейти в директорию с программой (>cd ..\TestTask)
3) Запустить скрипт сборки программы (BUILD.bat)

6) Ввести текст в командную строку
7) Запустить программу
8) Получить результат в командной строке
